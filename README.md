# Data

`bank-additional-full.csv` — UCI Bank Marketing dataset (41,188 rows, 21 columns).
Direct marketing (phone) campaigns of a Portuguese bank, 2008-2010.

Source: [UCI Machine Learning Repository — Bank Marketing](https://archive.ics.uci.edu/dataset/222/bank+marketing)

Citation: Moro, S., Cortez, P., & Rita, P. (2014). A Data-Driven Approach to
Predict the Success of Bank Telemarketing. Decision Support Systems.
